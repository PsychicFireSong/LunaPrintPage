# Luna's Print Page

A responsive website for showcasing sticker prints, photography, and artwork services for convention attendees.

## Features

### Home Page
- **Dynamic Welcome Title**: Rotates through 16 different languages every 3 seconds with smooth fade transitions
- **Responsive Navigation**: Desktop navigation bar with mobile hamburger menu
- **Three Main Sections**:
  - Introduction with portrait and bio
  - Contact information with convention details
  - Services overview with call-to-action buttons
- **Interactive Elements**: Image expansion modals, hover effects, scroll animations

### Stickers Page
- **Order Form**: Complete form for sticker submissions with validation
- **Material Information**: Details about 4 available sticker materials
- **FAQ Section**: Comprehensive answers to common questions
- **Upload Rate Limiting**: Prevents abuse with 5 uploads per 30 seconds limit
- **File Validation**: Maximum 6MB, JPEG/PNG only

## Technology Stack

- HTML5
- CSS3 (External stylesheet in Resources/styles.css)
- Vanilla JavaScript
- Responsive Design (Mobile-first approach)
- Vercel-ready deployment

## Deployment

This site is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.

### Deployment Steps:
1. Push code to GitHub
2. Connect repository to Vercel
3. Vercel will automatically detect the configuration
4. Site will be live at your Vercel URL
